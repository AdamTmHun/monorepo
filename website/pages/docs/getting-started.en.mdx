## VSCode extension (takes 60 seconds)

1. Install the VSCode extension [download link](https://marketplace.visualstudio.com/items?itemName=inlang.vs-code-extension).
2. Create an `inlang.config.json` file in your project.

   - Either run `npx @inlang/cli init` in the terminal (but node must be installed).
   - Or manually create the JSON config:  
     For an example and the config schema go to the [config reference](/docs/reference/config).

## Translation Management (takes 2 minutes)

1. Create a project in [the dashboard](https://app.inlang.dev).
2. Import your translations either via copy & paste or [the CLI](/docs/reference/cli).
